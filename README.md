# DeckCraft Reforged
A performance and ease-of-use modpack for the Steam Deck.
# ❓ But why DR?
In this modpack I included some more unknown-ish (promise im not trying to offend anyone) mods such as [ScalableLux](https://modrinth.com/mod/scalablelux) and [Faster Random](https://modrinth.com/mod/faster-random)
